# Vowel Consonants
Given console input, print number of vowels and consonants


## RUN THE CODE via 
1.  Use lex to create C file via : 
    ```bash
    flex vowel_consonants.l
    ```

2.  Use GCC to compile C file via :
    ```bash
    gcc lex.yy.c 
    ```

3. Run file with temp.txt as command line argument :
    ```bash
    ./a.out
    ```
